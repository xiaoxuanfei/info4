# 快速开始

## 环境要求

- 现代浏览器
- VS Code + [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)

## 启动步骤

1. 用 VS Code 打开 **`break-info` 根目录**
2. 安装 Live Server 扩展
3. 点击 **Go Live**
4. 访问 `http://127.0.0.1:8080`

## 数据

确保 `data/info4.json` 存在。首次运行若缓存为空，会自动从该文件加载。

## 常见问题

| 现象 | 解决 |
|------|------|
| 数据加载失败 | 用 Live Server 打开，不要直接双击 HTML |
| 修改未保存到代码 | info4 页导出 info4.json 覆盖 data/ |
